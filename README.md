# GPM Library Management System

GPM Library Management System is a real-time web-based application developed to simplify and automate library operations used in our college. It efficiently manages books, users, issuing, returning, and records tracking through a user-friendly interface.

## Features
- Add, update, delete books
- Manage users
- Issue and return books
- Search books
- Track records

## Tech Used
HTML, CSS, JavaScript

## Purpose
To reduce manual work and make library management faster and more efficient.
